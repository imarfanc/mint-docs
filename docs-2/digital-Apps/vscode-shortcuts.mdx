# custom shortcuts

- `cmd + alt + f1` - make interface minimal

# common shortcuts

- `ctrl + r` - open recent projects

# Basic Shortcuts

- `cmd + p` - open file
- `cmd + shift + p` - open command palette
- `cmd + k` - open command palette
- `cmd + k cmd + s` - open settings
- `cmd + k cmd + t` - open terminal
- `cmd + k cmd + r` - open search
- `cmd + k cmd + f` - open find
