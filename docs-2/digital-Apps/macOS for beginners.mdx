# Operating Systems

## macOS

macOS is a Unix-based operating system developed by Apple Inc. It serves as the primary OS for Apple's Macintosh computers, including the MacBook, iMac, and Mac Pro.

## Windows

Windows is a family of operating systems developed by Microsoft, serving as the primary OS for a vast range of personal and professional computers.


---

# Applications

## File Explorers

### Finder (macOS)

Finder is the default file manager for macOS, allowing users to navigate the file system and manage files and folders.

### File Explorer (Windows)

File Explorer is the default file manager for Windows, enabling users to browse and organize files and directories.

## Terminal

### Terminal (macOS)

Terminal provides a command-line interface for macOS, allowing users to execute commands and run scripts.

### Command Prompt / PowerShell (Windows)

Windows offers Command Prompt and PowerShell as command-line interfaces for executing commands and automating tasks.

## Web Browsers

### Safari (macOS)

Safari is Apple's default web browser for macOS, known for its speed and energy efficiency.

### Microsoft Edge (Windows)

Microsoft Edge is the default web browser for Windows, offering integration with Microsoft services and features.

### Chrome (Cross-platform)

Google Chrome is a widely-used web browser available on both macOS and Windows platforms.

### Firefox (Cross-platform)

Mozilla Firefox is an open-source web browser that supports various operating systems, including macOS and Windows.

## Text Editors

### VS Code

Visual Studio Code is a versatile text editor developed by Microsoft, supporting numerous programming languages and extensions.

### TextEdit (macOS)

TextEdit is a simple text editor for macOS, suitable for basic text editing tasks.

### Notepad (Windows)

Notepad is a basic text editor included with Windows, useful for simple text editing.

## اردو: آپریٹنگ سسٹمز

### macOS پر اردو زبان کا استعمال

macOS پر اردو میں لکھنے کے لیے:

1. **سسٹم سیٹنگز** کھولیں۔
2. **کی بورڈ** > **ان پٹ سورسز** پر جائیں۔
3. **ایڈ (+)** بٹن پر کلک کریں اور "Urdu" تلاش کریں۔
4. اردو کی بورڈ لے آؤٹ منتخب کریں اور **ایڈ** پر کلک کریں۔
5. ان پٹ مینو سے اردو کی بورڈ منتخب کریں یا **کنٹرول + اسپیس بار** دبائیں۔

بہتر اردو لکھائی کے لیے، *جمیل نوری نستعلیق* جیسے فونٹس انسٹال کریں۔ آن لائن دستیاب رہنماؤں سے مدد حاصل کی جا سکتی ہے۔

### Windows پر اردو زبان کا استعمال

Windows 10 یا 11 پر اردو زبان شامل کرنے کے لیے:

1. **سیٹنگز** > **ٹائم اینڈ لینگویج** > **لینگویج اینڈ ریجن** پر جائیں۔
2. **پریفرڈ لینگویجز** کے تحت **ایڈ اے لینگویج** پر کلک کریں۔
3. "Urdu" تلاش کریں اور منتخب کریں۔
4. **نیکسٹ** پر کلک کریں، پھر **انسٹال** پر کلک کریں۔
5. انسٹالیشن کے بعد، ٹاسک بار پر لینگویج بار سے یا **ونڈوز کی + اسپیس بار** دباکر زبان تبدیل کریں۔

اردو فونٹ اور کی بورڈ لے آؤٹ کے لیے **پاک اردو انسٹالر** استعمال کریں، جو اردو ٹائپنگ کو آسان بناتا ہے۔
